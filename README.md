# monkey-nim
monkey language made by Nim

## SetUp

Install nim language

```
$ make install
```

Setup Program

```
$ make init
```


## Usage

run monkey

```
$ make monkey
```


## Test

```
$ make test
```

or

```
$ make lexer_test
$ make token_test
$ make parser_test
$ make eval_test
$ make buildin_test
```
